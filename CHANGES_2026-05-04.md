# Changes — 2026-05-04

## 1. ADC Rate Test (`adc_rate_test.py`) — new file
Wrote a Python script to measure the actual ADC sampling rate on the board.
Connects via OpenOCD + ST-LINK, reads a counter from firmware twice (2 sec apart), and computes the rate.

**Result:** 1997 Hz actual — hardware can do ~88,000 Hz, so we're at 2% capacity.
**Why it matters:** confirmed we had room to crank up the control loop.
